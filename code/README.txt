Executables Usage:
In bin folder run all the executables and follow prompts.
You can also choose "n" for all depth, translate, rotate and scale.
This will lead to default values being picked.


Defaults are :
	depth 		= 2
	Translate 	= {2, 0, -4}
	Rotate		= {0, 0, 2}
	Scale 		= {1, 1, 1} 
	
What is implemented?

1. Wired cube, tetrahedron, octahedron, sphere(icosahedron), ellipse, torus, dodecahedron.
2. Shaded cube, tetrahedron, octahedron, sphere(icosahedron), ellipse, torus, dodecahedron.
3. RGB Shaded cube, tetrahedron, octahedron, sphere(icosahedron), ellipse, torus, dodecahedron.
4. Translate, rotate, scale/zoom on above figures
